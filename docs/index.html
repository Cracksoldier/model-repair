Fix broken 3D meshes
with model-repair

A CGAL-powered mesh repair library for Linux and Windows with a CLI and Qt6 GUI. Automatically fixes duplicate vertices, degenerate faces, non-manifold topology, bad normals, open holes, and self-intersections — in a single pass.

What it fixes

Seven repair steps, fully automated

Every step runs in a strict order — each is a precondition for the next. Enable or disable individual steps via CLI flags or GUI checkboxes.

🔗

Duplicate Vertices

Merges coincident points into shared vertices, converting raw triangle soup into a proper half-edge mesh. STL files always require this step.

Learn more →
📐

Degenerate Triangles

Removes zero-area faces with collinear or coincident vertices that corrupt all downstream topology analysis.

Learn more →
🔀

Non-Manifold Geometry

Splits vertices where more than two faces share an edge, restoring the disk topology required for watertight meshes.

Learn more →
🔃

Inconsistent Normals

Re-orients face windings per connected component so all triangles point outward, eliminating inside-out or inverted surfaces.

Learn more →
🕳️

Open Holes

Fills boundary cycles with smooth or flat triangulation, closing gaps left by upstream CAD export errors or boolean operations.

Learn more →
⚡

Self-Intersections

Heals overlapping and penetrating triangles using exact-arithmetic CGAL computation (EPECK kernel). Thorough but slow.

Learn more →
🖥️

3D Before/After Preview

Side-by-side OpenGL 3.3 comparison window with synchronized arcball camera, pan, and zoom — opens automatically after repair.

Learn more →
📦

Batch Processing

Drop multiple files into the GUI batch window, set an output directory, and repair them all with one click.

Learn more →
🔍

Diagnose Mode

Run the full detection pipeline without modifying the mesh. Inspect every issue before deciding whether to repair. Available as --diagnose on the CLI and a dedicated button in the GUI.

Learn more →
📏

Geometry Statistics

Surface area (mm²) and volume (mm³) are computed before and after repair and shown in every report. Volume is reported only for closed (watertight) meshes.

Learn more →
〰️

Mesh Smoothing

Cotangent-weighted feature-preserving Laplacian with crease barriers. Runs multi-core on CPU (TBB) or entirely on GPU (Vulkan compute) — vertex data stays in VRAM across all iterations, with no per-iteration round-trips.

Learn more →
🔺

Mesh Decimation

Reduce polygon count after repair using CGAL Surface_mesh_simplification. The intermediate repaired mesh is saved automatically before decimation is applied.

Learn more →
🧹

Internal Geometry Removal

Detects and removes faces hidden inside a closed mesh using CGAL's Side_of_triangle_mesh inside-test. Eliminates redundant shell fragments that cause slicer artefacts.

Learn more →
🔷

Subdivision

Increase mesh resolution with Loop (triangles) or Catmull-Clark (general) subdivision — a quality-preserving alternative to remeshing for smooth organic shapes.

Learn more →
🌡️

Wall Thickness Heatmap

AABB ray-cast analysis colors each face by measured wall thickness — blue for thin walls, red for thick — so potential print failures are visible at a glance.

Learn more →
📐

Overhang Highlight

Shades faces by overhang angle in the Before/After preview. Steep overhangs appear red; well-supported horizontal faces appear blue — no support estimator needed.

Learn more →
🐚

Shell Separation

Detects disconnected mesh shells and lets you keep the largest, or export every shell as a separate file — useful for multi-body imports and split models.

Learn more →
Supported formats: STL binary STL ASCII OBJ 3MF PLY GLB glTF

Quick Setup

Install or build

The easiest way to get started is a pre-built release — no installation required. Choose your platform below, or build from source.

# Set the version tag from the Releases page, then download
VERSION=v0.4.0
wget https://github.com/Cracksoldier/model-repair/releases/download/${VERSION}/model-repair-${VERSION}-x86_64.AppImage
chmod +x model-repair-${VERSION}-x86_64.AppImage

# Launch the GUI
./model-repair-${VERSION}-x86_64.AppImage

# Or run the CLI
./model-repair-${VERSION}-x86_64.AppImage --appimage-extract-and-run \
  broken.stl repaired.stl --verbose

The AppImage bundles Qt 6, libmodelrepair, and all dependencies. It runs on any x86-64 Linux with glibc ≥ 2.17. FUSE is not required — the AppImage self-extracts automatically. Download links for each release are on the GitHub Releases page. 

# Download the ZIP from the Releases page and extract it
# https://github.com/Cracksoldier/model-repair/releases/latest

# The extracted folder is self-contained — no installer, no PATH changes needed.
# Double-click model-repair-gui.exe to open the GUI.
# Run the CLI from a terminal:
model-repair.exe broken.stl repaired.stl --verbose

The Windows ZIP bundles Qt 6, all MSYS2/UCRT64 runtime DLLs (GMP, MPFR, GCC runtime, libwinpthread), and libmodelrepair — nothing to install. Extract anywhere and run. Built with MinGW-w64 GCC via MSYS2 UCRT64. Requires 64-bit Windows 10 or later. 

# Install system dependencies (includes TBB + Vulkan GPU smoothing)
sudo pacman -S cgal eigen3 boost gmp mpfr qt6-base \
  onetbb vulkan-headers vulkan-icd-loader shaderc cmake git
# Install system dependencies (TBB included)
sudo apt install libcgal-dev libeigen3-dev libboost-dev libgmp-dev \
  libmpfr-dev qt6-base-dev libtbb-dev cmake build-essential git

# Optional: Vulkan GPU smoothing
sudo apt install libvulkan-dev glslc
# Install system dependencies (TBB included)
sudo dnf install cgal-devel eigen3-devel boost-devel gmp-devel \
  mpfr-devel qt6-qtbase-devel tbb-devel cmake gcc-c++ git

# Optional: Vulkan GPU smoothing
sudo dnf install vulkan-headers vulkan-loader-devel glslc
# Clone and build
git clone https://github.com/Cracksoldier/model-repair.git
cd model-repair
cmake --preset release          # downloads lib3mf, spdlog, CLI11, Catch2
cmake --build build/release -j$(nproc)
# Binaries: build/release/cli/model-repair  and  build/release/gui/model-repair-gui
# Downloads: lib3mf, tinygltf, CLI11, spdlog, Catch2
Run the CLI
build/release/cli/model-repair \
  broken.stl repaired.stl --verbose

Run the GUI
build/release/gui/model-repair-gui

# Drop an STL, OBJ, 3MF, PLY, GLB, or glTF
# onto the window, adjust options,
# click Repair. A 3D before/after

Need a no-Qt build? Use cmake --preset release-nogui to build only libmodelrepair.so and the CLI. See Architecture for details on running tests and embedding the library.
